# BLOOP Theme for Vim

High contrast theme with a minimal, modern color palette.

- dark theme only atm,
- will have light theme
- true type support
- 256 colors support
- includes colors for lightline and fzf
- includes colors for lightline and fzf
- does not set a background color, so that using opacity still works.
